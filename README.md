## MONARCH Template for NamelessMC (v2)

How to install:

    Upload to Main Directory.
    Login to AdminCP.
    Go to Styles.
    Click "Install" > Find "Monarch" and Activate.

That's it.

You're allowed to:
- Rewrite it

You're not allowed to:
- Sell it
- Give it out as your own
